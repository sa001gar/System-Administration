1. Internal Components of a PC

A Personal Computer (PC) contains several important internal components that allow it to function efficiently:

  • Motherboard: The central hub connecting all hardware components.
  • CPU (Central Processing Unit): The brain of the computer that processes instructions.
  • RAM (Random Access Memory): Temporary memory used for active processes.
  • Storage (SSD/HDD): Long-term storage for the operating system and files.
  • GPU (Graphics Processing Unit): Renders images, video, and animations.
  • PSU (Power Supply Unit): Provides power to the computer components.
  • Cooling System: Keeps the CPU and other components from overheating.

2. Steps to Assemble a PC

Assembling a PC involves several steps to ensure all components work together properly:

  1. Prepare your workspace by grounding yourself to avoid static electricity.
  2. Install the CPU into the motherboard and apply thermal paste before adding the cooler.
  3. Insert the RAM into the motherboardâ€™s RAM slots.
  4. Secure the motherboard inside the case.
  5. Install the Power Supply Unit (PSU) and connect power cables to the motherboard and drives.
  6. Install storage drives (HDD/SSD) and connect data cables to the motherboard.
  7. Mount the Graphics Processing Unit (GPU) in the PCIe slot.
  8. Connect all case fans and other peripherals to the motherboard.
  9. Power on the system and install the operating system.

3. Network Connection Between Two PCs

To establish a network connection between two PCs in Windows:

  1. Open Control Panel and navigate to Network and Sharing Center.
  2. Select Change adapter settings.
  3. Right-click the network adapter and select Properties.
  4. Choose Internet Protocol Version 4 (TCP/IPv4) and enter a static IP address for both PCs.
  5. Ensure both PCs are on the same subnet and save the changes.

4. UNIX Commands and Their Functions

  • Terminal: Interface to interact with the system using commands.
  • Shell: Command interpreter (e.g., Bash).
  • cat: Display content of files.
  • ls: List directory contents.
  • cd: Change directory.
  • date: Display system date and time.
  • cal: Display a calendar.
  • man: View command manual pages.
  • echo: Display a line of text.
  • pwd: Print current working directory.
  • mkdir: Create a new directory.
  • rm: Remove files.
  • rmdir: Remove empty directories.
  • ps: Display running processes.

5. Sharing Printers and Files Over a Network

To share a printer or files over a network:

  1. Open Control Panel and go to Network and Sharing Center.
  2. Enable File and Printer Sharing.
  3. Right-click on the file or folder and select Properties.
  4. Go to the Sharing tab and click Share.
  5. Select the users you want to share with and set permissions.

6. Creating Users in Windows

Follow these steps to create new users in Windows:

  1. Open Control Panel and go to User Accounts.
  2. Select Manage another account and click Add a new user.
  3. Choose whether the new user should be an Admin or a regular user.
  4. Complete the setup and save the new user profile.

7. Creating a Remote Desktop Connection

To set up a remote desktop connection:

  1. Right-click on This PC and go to Properties.
  2. Select Remote Settings and enable Allow remote connections to this computer.
  3. Open Remote Desktop Connection and enter the IP address of the target computer.
  4. Click Connect to establish a remote session.
